# AI GameMaster Frontend

React Native приложение на Expo.

## Требования

- **Node.js** 18+
- **npm**
- **Expo Go** на телефоне — [iOS](https://apps.apple.com/app/expo-go/id982107779) / [Android](https://play.google.com/store/apps/details?id=host.exp.exponent)

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

После запуска в терминале появится QR-код. Откройте **Expo Go** на телефоне и отсканируйте его — приложение загрузится на устройство.

> Телефон и компьютер должны быть в одной Wi-Fi сети.

## Команды

| Команда | Описание |
|---------|----------|
| `npm start` | Запуск dev-сервера (QR-код для Expo Go) |
| `npm run android` | Запуск на Android-эмуляторе/устройстве |
| `npm run ios` | Запуск на iOS-симуляторе/устройстве |
| `npm run web` | Запуск в браузере |

## Горячие клавиши в dev-сервере

После `npm start` в терминале доступны:

- `a` — открыть на Android
- `i` — открыть на iOS-симуляторе
- `w` — открыть в браузере
- `r` — перезагрузить приложение
- `j` — открыть дебаггер
